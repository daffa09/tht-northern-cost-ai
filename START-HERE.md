# Start Here — Northern Coast AI Platform Engineer Take-Home

Welcome. This is a 3-test technical assessment. About **3 hours total** if you work with AI tools daily. Less if you're fluent. Significantly more if you're not — and that itself is a signal.

## Quickest way to start

**Open this entire folder in your AI coding tool** — Claude Code, Cursor, Codex, Windsurf, whatever you use daily.

Then paste this prompt into your AI:

```
This is a hiring take-home for a senior AI Platform Engineer role at Northern Coast Beverages. There are 3 tests in subfolders.

Please:
1. Read README.md at this folder's root
2. Read each test's BRIEF.md (in test-1-build/, test-2-research/, test-3-critique/)
3. Summarize what I need to deliver across the three tests
4. Note any constraints I should pay attention to
5. Ask me which test I want to start with

We'll work through them together. Treat this like real production work — help me think, but don't substitute for my judgment.
```

Then start the conversation from there.

If you don't have a tool you're comfortable using this way, you might find this assessment difficult. **That's by design** — the role is for someone who directs AI tools fluently. Not knowing how to use these tools well is itself a signal we calibrate against.

## What you'll find in this folder

```
.
├── START-HERE.md            ← you are here
├── README.md                ← detailed overview of the assessment
├── test-1-build/
│   └── BRIEF.md             ← build a deployed scoring endpoint
├── test-2-research/
│   └── BRIEF.md             ← open-problem research + recommendation
└── test-3-critique/
    ├── BRIEF.md             ← critique an RFC from a previous candidate
    └── RFC-TO-CRITIQUE.md   ← the RFC you're evaluating
```

## What to submit

Place your work in a `submission/` subfolder inside each test. When done, zip the whole top-level folder (with your `submission/` folders inside) and email it back.

## Time

- **3 hours total** if you're fluent with AI tools
- **5-6 hours** if you're competent but not daily-fluent
- **10+ hours** if these tools are new to you — and that's likely a signal this role isn't the right fit yet

Take what you need. **Track your time honestly** — we discuss it in the debrief.

## How we'll evaluate

We read your submissions for **decisions and reasoning** more than for output volume. The debrief afterward (60-90 min) is where the hire decision actually happens. The tests get you to the conversation.

Good luck.
